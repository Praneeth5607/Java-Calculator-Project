# Java-Calculator-Project

This is a Java-based calculator project that performs different operations like Arithmetic, Temperature Conversion, Mass Conversion, Length Conversion, and Time Conversion.
Each feature is written in a separate Java file to make the project modular and easy to maintain.

Project Structure:-
-> Main_Menu.java
-> Arithmetic_Menu.java
-> Temperature_Menu.java
-> Mass_Menu.java
-> Length_Menu.java
-> Time_Menu.java

## How to Run

1. Download or clone the repository.
2. Open the project in any Java IDE (IntelliJ, Eclipse, VS Code).
3. Compile all `.java` files.
4. Run `Main_Menu.java` to start the calculator.
